WHY STATIC

GitHub Pages 안내

GitHub Pages는 정적 호스팅입니다. Flask 서버, PDF 렌더링, Gemini API 호출 프록시는 Pages에서 직접 실행되지 않습니다.

SECURITY

API 키 비공개

API 키는 저장소와 Pages에 포함하지 않습니다. 로컬 GUI 입력칸, local_settings.json, 또는 GEMINI_API_KEY 환경변수로만 사용합니다.

RUN

로컬 실행

  • pip install -r requirements.txt
  • python3 app.py
  • http://127.0.0.1:5057/
