Auto Analysis 처음 실행 안내

1. 압축 풀기
- ZIP 파일을 바탕화면이나 문서 폴더처럼 찾기 쉬운 곳에 풀어 주세요.
- 압축 파일 안에서 바로 실행하지 말고, 반드시 폴더로 풀어 놓은 뒤 실행하세요.

2. Windows에서 실행
- "Auto Analysis 실행 - Windows.bat"을 더블클릭하세요.
- 처음 실행할 때는 필요한 파일을 자동 설치하므로 1-3분 정도 걸릴 수 있습니다.
- 검은 창은 서버 창입니다. 사용하는 동안 닫지 마세요.
- 브라우저가 자동으로 열리지 않으면 http://127.0.0.1:5057/ 을 직접 열면 됩니다.
- Python을 찾지 못한다는 메시지가 나오면 Python 3.11 이상을 설치하고 다시 실행하세요.
- 이미 Python을 설치했다면 Python 설치 프로그램을 다시 실행한 뒤 Modify > Add python.exe to PATH를 선택하세요.

3. macOS에서 실행
- "Auto Analysis 실행 - macOS.command"를 더블클릭하세요.
- 보안 경고가 뜨면 파일을 우클릭한 뒤 "열기"를 누르세요.
- 처음 실행할 때는 필요한 파일을 자동 설치하므로 1-3분 정도 걸릴 수 있습니다.

4. Gemini API 키
- 이 공유본에는 Gemini API 키가 들어 있지 않습니다.
- 각 사용자가 화면의 "Gemini API 키" 입력칸에 자기 키를 직접 넣어 사용하면 됩니다.
- 팀원에게 보낼 때도 API 키 파일이나 키 문구를 따로 넣지 마세요.

5. 샘플 열기
- 화면 아래쪽의 "모의고사 보관함"에서 "4월 NEW"를 찾으세요.
- "불러오기"를 누르면 샘플 문항 이미지와 해설을 확인할 수 있습니다.

6. 종료
- Windows에서는 검은 서버 창에서 Ctrl+C를 누르거나 창을 닫으면 됩니다.
- macOS에서는 터미널 창에서 Ctrl+C를 누르거나 창을 닫으면 됩니다.
