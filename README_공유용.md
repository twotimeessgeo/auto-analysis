# Auto Analysis 공유용

이 폴더는 팀원이 바로 실행할 수 있도록 정리한 Auto Analysis 공유본입니다.
Gemini API 키는 포함되어 있지 않습니다.

## Windows

1. ZIP 압축을 먼저 풉니다.
2. `Auto Analysis 실행 - Windows.bat`을 더블클릭합니다.
3. 처음 실행할 때 필요한 패키지가 자동 설치됩니다. 1-3분 정도 기다리면 됩니다.
4. 브라우저가 열리면 `Gemini API 키` 입력칸에 자기 키를 입력합니다.

Python을 찾지 못한다는 메시지가 나오면 Python 3.11 이상을 설치한 뒤 다시 실행하세요.
설치할 때 `Add python.exe to PATH`를 체크하면 가장 쉽습니다.
이미 Python을 설치했다면 설치 프로그램을 다시 실행한 뒤 `Modify > Add python.exe to PATH`를 선택하세요.

## macOS

1. ZIP 압축을 먼저 풉니다.
2. `Auto Analysis 실행 - macOS.command`를 더블클릭합니다.
3. 보안 경고가 뜨면 파일을 우클릭한 뒤 `열기`를 누릅니다.
4. 브라우저가 열리면 `Gemini API 키` 입력칸에 자기 키를 입력합니다.

## 샘플

`모의고사 보관함`에서 `4월 NEW`를 불러오면 샘플 문항 이미지와 해설을 확인할 수 있습니다.

## 종료

서버 창을 닫거나 `Ctrl+C`를 누르면 종료됩니다.
