# Graph And Exam Guide

이 문서는 TWOTIMESS 스타일을 유지하면서, 기후·통계 데이터를
`그래프`, `표`, `문항형 비교 화면`으로 어떻게 써야 하는지 정리한 실전 가이드입니다.

## 1. 기본 원칙

1. 그래프는 "예쁘게"보다 "읽히게"가 우선입니다.
2. 제목에서 이미 설명한 항목은 축에서 반복하지 않습니다.
   - 예: 제목이 `월 평균 기온`이면 축은 `기온 (°C)`가 아니라 `(°C)`만 둡니다.
   - 예: 제목이 `누적 강수량`이면 축은 `(mm)`만 둡니다.
3. 비교 그래프는 가능한 한 `공통 축`을 써야 합니다.
4. 하나의 화면에 표와 그래프가 같이 있으면, 같은 수치를 다른 방식으로 확인할 수 있어야 합니다.
5. 장식성 요소보다 `축`, `범례`, `기준`, `각주`가 더 중요합니다.

## 2. 어떤 통계를 기본으로 보여줄지

### 세계지리

항상 우선적으로 보여줄 값:

- 연평균 기온
- 연강수량
- 1월 평균 기온 / 강수량
- 7월 평균 기온 / 강수량
- 최난월 기온
- 최한월 기온
- 연교차
- 월별 기온
- 월별 강수량
- 누적 강수량

상황에 따라 추가할 값:

- 반구
- 대륙
- 쾨펜 기후 구분
- 위도 / 경도
- 고도

### 한국지리

항상 우선적으로 보여줄 값:

- 1월 평균 기온
- 8월 평균 기온
- 겨울(12~2월) 평균 기온
- 여름(6~8월) 평균 기온
- 1월 강수량
- 8월 강수량
- 겨울 강수량 합
- 여름 강수량 합
- 연교차
- 월 평균 기온
- 누적 강수량

가능하면 같이 보여줄 값:

- 무상일수
- 열대야일수
- 서리일수 또는 결빙 관련 지표
- 지역 구분(남한 / 북한 / 권역)

## 3. 그래프 종류별 사용 기준

### 월별 기후 그래프

용도:

- 한 지역의 계절 변화를 읽게 할 때
- 건기 / 우기 / 계절 역전 여부를 보여줄 때

형식:

- 기온: 선 그래프
- 강수량: 막대 그래프
- 한 카드 안에서 기온과 강수량을 함께 배치

주의:

- 막대와 선의 축이 다르다는 걸 명확히 보여줘야 합니다.
- 막대 radius는 주지 않습니다.

### 월 평균 기온 비교 그래프

용도:

- 여러 지역의 기온 연중 변화를 한 번에 비교할 때
- 반구 차이, 해양성/대륙성, 고도 차이를 읽게 할 때

형식:

- 다중 선 그래프
- 선택한 지역 전체에 같은 y축 사용

주의:

- 각 지역마다 축이 다르면 비교 그래프로서 가치가 떨어집니다.
- 선 스타일은 3~4종 이내에서만 변형합니다.

### 누적 강수량 그래프

용도:

- 강수 집중 시기
- 우기의 시작과 강수 누적 속도 차이
- 몬순형 / 연중형 / 겨울우세형 패턴 비교

형식:

- 누적 선 그래프
- 선택한 지역 전체에 같은 y축 사용

주의:

- "누적"이라는 말이 제목에 있으면 축에는 `(mm)`만 둡니다.
- 각주는 반드시 "1월부터 해당 월까지의 합"이라고 밝혀야 합니다.

### 편차 그래프

용도:

- 선택 지역 평균 대비 차이
- 특정 기준 지역 대비 차이
- 시험 문제식 "어느 지역이 더 높고 낮은가" 비교

형식:

- 기온: 점 그래프 또는 얇은 막대
- 강수량: 막대 그래프

주의:

- 기준이 평균인지 특정 지역인지 반드시 표시합니다.
- 특정 지역 기준이면 그 기준 지역은 편차 그래프에서 제외하는 것이 읽기 쉽습니다.

### 연교차 그래프

용도:

- 대륙성 / 해양성
- 내륙 / 해안
- 위도 차이보다 계절 대비가 큰 지역을 찾게 할 때

형식:

- 단일 막대 그래프

주의:

- 제목이 `연교차 비교`이면 축 상단에는 `(°C)`만 둡니다.
- 그래프 안에서 "8월 평균기온 - 1월 평균기온"을 반복 표기하지 않습니다.

## 4. 축, 범례, 각주 작성 규칙

### 축

- 제목이 이미 설명한 정보는 축에서 반복하지 않습니다.
- 축 상단 라벨은 가급적 짧게 둡니다.
  - 좋은 예: `(°C)`, `(mm)`
  - 나쁜 예: `기온 편차 (°C)`를 제목과 축에 둘 다 쓰기

### 범례

- 지역명은 `A, B, C`보다 실제 지역명을 우선합니다.
- 지역 수가 많지 않으면 범례보다 선 끝 라벨을 고려합니다.
- 기호/선형/색 차이는 최소한만 씁니다.

### 각주

- 1줄이면 `*`
- 2줄이면 `*`, `**`
- 3줄이면 `*`, `**`, `***`
- 각주는 설명이 아니라 `해석 조건`을 적는 곳입니다.

각주에 적기 좋은 내용:

- 기준 기간
- 편차 산식
- 누적 산식
- 평균 기준 또는 참조 지역 기준
- 적용한 공통 축 여부

## 5. 시험형 화면 설계 기준

### 공통

- 한 화면에 너무 많은 지역을 넣지 않습니다.
  - 기본은 4개
  - 많아도 5개 정도
- 지역 조합은 "구분 가능성"이 있어야 합니다.
  - 너무 비슷한 4개는 피합니다.
  - 그렇다고 지나치게 극단적인 4개만 고정하지도 않습니다.

### 세계지리 문제용 조합

좋은 조합 예:

- 적도우림 / 사바나 / 지중해성 / 냉대
- 서안해양성 / 지중해성 / 스텝 / 열대몬순
- 남반구 / 북반구가 섞인 조합

피해야 할 조합:

- 모두 비슷한 위도와 비슷한 해양성 기후
- 모두 강수량이 비슷해서 패턴이 겹치는 조합

### 한국지리 문제용 조합

좋은 조합 예:

- 동해안 / 서해안 / 내륙 / 북부 고위도
- 남부 해안 / 중부 내륙 / 북부 내륙 / 산지 인접

피해야 할 조합:

- 서로 너무 가까운 4지점
- 반대로 항상 가장 먼 4지점만 고정되는 조합

권장:

- 랜덤 선택 시 "너무 가깝지만 않음" 정도의 거리 제한만 둡니다.

## 6. 시험형 문항 아이디어

### 세계지리

- 1월과 7월 중 어느 지역이 남반구인지 찾기
- 건기와 우기가 바뀌는 지역 찾기
- 연교차가 가장 큰 지역 찾기
- 최난월 기온과 최한월 기온으로 대륙성/해양성 판단하기
- 누적 강수량 기울기로 몬순 시기 찾기

### 한국지리

- 1월/8월 기온으로 북부-남부 차이 읽기
- 여름/겨울 강수 편차로 계절 강수 집중성 비교하기
- 연교차로 내륙성 정도 비교하기
- 무상일수와 열대야일수로 남북/해안-내륙 차이 해석하기

## 7. 표와 그래프를 함께 둘 때의 규칙

1. 카드 상단에는 핵심 통계
2. 카드 중간에는 그래프
3. 카드 하단에는 상세 표

이 순서를 추천합니다.

이유:

- 사용자는 먼저 지역의 성격을 짧게 파악하고
- 다음에 패턴을 보고
- 마지막에 월별 수치를 확인하기 때문입니다.

## 8. Codex에게 추가로 전달하면 좋은 문장

아래 문장은 다른 Codex에게 그대로 보내도 됩니다.

```text
그래프는 시험형 비교에 바로 쓸 수 있게 설계해주세요. 제목에서 이미 설명한 항목은 축에서 반복하지 말고 단위만 남겨주세요. 다지역 비교 그래프는 반드시 공통 축을 사용하고, 표와 그래프가 같은 수치를 서로 다른 방식으로 확인하는 구조가 되게 해주세요. 세계지리는 1월/7월 중심, 한국지리는 1월/8월과 겨울/여름 중심으로 정리해주세요.
```
