## 1. 자연지리와 무엇이 다르고 무엇을 유지할지

### 그대로 유지할 것

- Pretendard 기반 타이포
- 흑백 + 종이색 배경
- 직각 패널과 1px 보더
- 큰 제목, 작은 보조설명
- 과한 그림자, 그라데이션, 장식 제거
- 표와 그래프를 함께 두는 정보성 레이아웃

### 바꿔야 할 것

- 월별 기온/강수 중심 카드
- 기후대 필터 중심 구조
- 계절 편차 위주의 비교 패널

### 인문지리에서 대신 중심이 될 것

- 국가/대륙/권역 단위의 `구성비`
- `시점 간 변화`
- `순위 비교`
- `산업 구조`
- `인구 이동`
- `도시화`
- `자원/에너지`
- `종교/문화 구성`

즉, 자연지리는 "계절 패턴을 읽는 UI"였다면,
인문지리는 "구조와 변화, 관계를 읽는 UI"가 되어야 합니다.

## 2. 인문지리에서 우선 사용할 그래프 유형

레퍼런스를 기준으로 보면 아래 네 가지가 핵심입니다.

### 1. 수평 막대 비교 그래프

예:

- 대륙별 도시 수 변화
- 국가별 생산량 비교
- 지역별 에너지 소비량 비교

언제 쓰나:

- 항목명이 길 때
- 순위와 크기 비교가 중요할 때
- 두 시점 또는 두 지표를 나란히 보여줄 때

규칙:

- 기본은 `수평 막대`
- 두 시점 비교는 `연한 회색 / 진한 회색` 두 막대로
- 항목명은 왼쪽, 단위는 x축 끝에
- 단위 표기는 축에만 두고 제목에 반복하지 않음

### 2. 소형 다중 시계열 그래프

예:

- 출생률 / 사망률 / 순이동률
- 인구 증가율 변화
- 자원 생산 변화

언제 쓰나:

- 지역 또는 국가를 여러 개 한 화면에 비교할 때
- 추세는 같지만 폭이 다른 경우
- 시험형으로 "(가)(나)(다)를 구분하라" 구조가 필요할 때

규칙:

- 동일 축 사용
- 패널별 축 범위는 가급적 통일
- 선 스타일은 `실선`, `점선`, `가는 실선` 정도만
- 범례보다 선의 역할 차이를 먼저 읽히게

### 3. 100% 누적 막대 그래프

예:

- 종교 구성
- 산업 구조 비중
- 에너지원 비중
- 작물 비중

언제 쓰나:

- 합이 100%인 구성비 문제
- "어느 항목이 다수인지"와 "두 번째 비중이 무엇인지"가 핵심일 때

규칙:

- x축은 `0~100`
- 색 대신 `회색 농도 + 패턴` 조합 사용 가능
- 범례는 그래프 아래 한 줄 정렬
- 패턴 개수는 4개 내외로 제한

### 4. 도넛 / 원형 구성 그래프

예:

- 산업 구조와 GDP를 동시에 보여주기
- 중심값 + 구성비를 한 번에 보여주기

언제 쓰나:

- 중심의 절대량이 중요할 때
- 구성비 항목 수가 3~4개로 단순할 때

주의:

- 항목이 많으면 피합니다.
- 순위 비교만 중요하면 막대그래프가 더 낫습니다.
- 원형은 "예쁘다"보다 "중심값을 같이 담아야 한다"는 이유가 있을 때만 사용합니다.

## 3. 인문지리용 화면 구조 권장안

### A. 주제 개요 화면

- hero
- 핵심 통계 카드 3~5개
- 대표 비교 그래프 1~2개
- 지역/주제 필터

적합 주제:

- 도시화
- 에너지
- 인구 구조

### B. 시험형 비교 화면

- 상단: 문항 맥락 설명
- 중단: 표 + 그래프
- 하단: 비교용 각주

적합 주제:

- 국가/대륙 비교
- (가)(나)(다) 유형 구분
- 시점 변화 추론

### C. 지역별 상세 화면

- 지역명 / 국가명
- 대표 통계 배지
- 구성비 그래프
- 변화 그래프
- 설명용 표

적합 주제:

- 특정 국가 사례
- 특정 권역 사례
- 생산 구조 / 종교 / 인구 이동

## 4. 자연지리 카드 구조를 인문지리로 바꾸는 방법

자연지리 카드:

- 지역명
- 기후 분류
- 월별 기온/강수
- 기후 그래프
- 월별 표

인문지리 카드:

- 지역명 또는 주제명
- 핵심 속성 태그
- 대표 통계 3~5개
- 구성비 또는 변화 그래프
- 보조 표

예:

- `브라질`
- `남아메리카`, `신흥 공업`, `도시화`
- `도시화율`, `1차/2차/3차 산업 비중`, `순이동률`
- `산업구조 도넛 + 도시 인구 변화 선그래프`
- `연도별 표`

## 5. 인문지리 필터 설계 기준

자연지리처럼 기후 코드 필터를 앞세우지 말고, 아래 기준을 우선합니다.

- 대륙
- 국가
- 권역
- 주제
- 시점
- 산업 유형
- 인구 구조 유형
- 자원 / 에너지 유형

좋은 필터 예:

- `대륙`
- `선진국 / 개발도상국`
- `에너지 생산 / 소비`
- `도시화 / 인구 이동 / 종교 / 산업 구조`

## 6. 그래프 표현 톤

레퍼런스를 보면 인문지리 그래프는 자연지리보다 더 "문항 자료"에 가까워야 합니다.

그래서 다음을 권장합니다.

- 채도 높은 색 금지
- 회색 단계 3~4개 이내
- 필요할 때만 해칭 패턴 사용
- 선 굵기는 일정하게
- 격자선은 얇고 연하게
- 테두리는 분명하되 과하지 않게
- 범례는 하단 정렬

## 7. 시험형 문제에 특히 잘 맞는 구성

### 구성비 문제

- 100% 누적 막대
- 도넛
- 파이보다는 도넛 또는 누적 막대 우선

### 시점 변화 문제

- 두 시점 막대 비교
- 시계열 선그래프

### 지역 구분 문제

- 소형 다중 그래프
- 동일 축을 가진 카드 반복

### 국가/대륙 비교 문제

- 수평 막대
- 표 + 막대 병치

## 8. 자연지리와 공통으로 유지할 문장 습관

- 제목은 짧고 명확하게
- 축은 단위 중심
- 각주는 산식이나 해석 조건만 적기
- 본문 설명은 짧게
- 카드 안에서 시선 흐름은 `핵심 수치 -> 그래프 -> 표`
